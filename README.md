# MaiMei

## Описание приложения

MaiMei — это веб-приложение для управления коллекцией косметических продуктов. Позволяет пользователям вести каталог своей косметики, а для админов добавлять новые продукты, редактировать информацию об актуальных товарах (в том числе при помощи ИИ) и удалять устаревшие позиции. Приложение использует современный стек технологий: React с TypeScript, Redux Toolkit для управления состоянием и Supabase как Backend-as-a-Service платформу.

## Как получить исходники

```bash
git clone https://github.com/pr0stSasha/cosmetics-app.git
cd cosmetics-app
```
```Требования
Для запуска проекта локально вам понадобится:

Node.js: версия 18 или выше (рекомендуется LTS). Скачать Node.js

npm: обычно устанавливается вместе с Node.js

Учетная запись Supabase: для работы с базой данных и аутентификацией. Создать бесплатный проект в Supabase

(Опционально) Supabase CLI: необходим для локальной разработки Edge Functions. Установить Supabase CLI
```
```Запуск проекта
1. Установка зависимостей
Перейдите в директорию проекта и выполните:

bash
npm install
2. Настройка переменных окружения
В корне проекта создайте файл .env на основе примера (если есть .env.example):

bash
cp .env.example .env
Или создайте файл вручную:

bash
touch .env
Откройте файл .env и заполните следующие переменные, используя данные из панели управления вашего проекта в Supabase (Settings > API):

env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
Где взять значения:

VITE_SUPABASE_URL → Project Settings > API > Project URL

VITE_SUPABASE_ANON_KEY → Project Settings > API > Project API Keys > anon public

3. Настройка базы данных (если требуется)
Если вы используете миграции или начальные скрипты:

bash
# Запуск скрипта инициализации БД (если есть)
npm run db:setup

# Или применение миграций
npm run db:migrate
4. Запуск приложения для разработки
Выполните команду:

bash
npm run dev
Приложение будет доступно по адресу: http://localhost:5173 (или другому порту, указанному в терминале).
```
Основной бизнес-кейс (Работа с каталогом косметики)
```Просмотр каталога и до
Откройте приложение в браузере
На главной странице отобразится список всех косметических продуктов, рекомендованных с учетом настроек в профиле
Продукты можно добавить в избранное, а также удалить оттуда
При изменении настроек в профиле раздел рекомендаций изменится с учетом Ваших параметров
```
```Для админов
1. Добавление нового продукта при помощи ИИ
Вставьте название продукта в соответствующую ячейку
Нажмите на кнопку генерации
Добавьте ссылку на фото продукта в соответствующую ячейку
Нажмите "Сохранить" - продукт появится в общем каталоге лдя всех пользователей, кому он подходит

2. Добавление нового продукта вручную
Настройте соотетствующие параметры
Нажмите "Сохранить" - продукт появится в общем каталоге лдя всех пользователей, кому он подходит

3. Редактирование продукта
В карточке любого продукта нажмите кнопку "Редактировать" (иконка карандаша)
В открывшейся форме измените нужные поля
Нажмите "Обновить" для сохранения изменений

4. Удаление продукта
В карточке продукта нажмите кнопку "Удалить" (иконка корзины)
Подтвердите действие во всплывающем окне
Продукт будет удален из каталога
```
